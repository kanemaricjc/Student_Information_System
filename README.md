# Student Information System

A Student info system for managing student records.

### Setup
1. Import `database_backup.sql` into your local MySQL.
2. Rename `db.properties.example` to `db.properties`.
3. Enter your local MySQL credentials in `db.properties`.
4. Run the project in NetBeans.

I modified the given files by Sir Jamie Rosal by a bit so that my root password isnt exposed to the public.
